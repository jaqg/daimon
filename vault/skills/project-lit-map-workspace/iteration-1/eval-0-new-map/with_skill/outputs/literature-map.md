# Literature Map — test project

*5 papers across 3 themes.*

## DFT adsorption methodology

- **[[smith2023-zeolite-adsorption]]** (A. Smith, 2023)
  Core PBE-D3 workflow for E_ads in BEA/MFI; E_ads range -0.2 to -1.4 eV provides reference values for our polymer calculations.
- **[[wu2024-dft-benchmark]]** (I. Wu, 2024)
  Benchmarks PBE-D3 vs experiment; shows bare PBE underestimates by 30-50%, directly justifying our dispersion correction choice.

## Zeolite characterisation and GCMC

- **[[chen2022-pore-geometry]]** (C. Chen, 2022)
  Introduces d_min metric for accessible pore diameter — the key classifier for whether a polymer fragment fits a given zeolite channel.
- **[[liu2021-gcmc-polymer]]** (E. Liu, 2021)
  GCMC with COMPASS III + Peng-Robinson fugacity; validates the force field and methodology we use for adsorption isotherms.

## Environmental context and motivation

- **[[brown2020-microplastic-review]]** (G. Brown, 2020)
  PE and PP dominant in environmental samples; supports introduction framing zeolite adsorption as a remediation strategy. (low relevance)
